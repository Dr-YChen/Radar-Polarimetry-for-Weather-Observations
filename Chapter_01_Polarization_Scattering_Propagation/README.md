# 第1章：电磁波极化、散射与传播

## 章节简介

本章探讨电磁波在大气中传播的基本原理，以及电磁波与单个水凝物粒子相互作用的物理机制。

**核心主题：**
- 电磁波偏振态的数学描述（Jones矢量、Stokes参数）
- 单粒子散射理论（瑞利散射、米散射）
- 电磁波在大气中的传播效应（衰减、相移）

## 关键公式

### Stokes 参数
$$[I, Q, U, V] = \begin{pmatrix} |E_h|^2 + |E_v|^2 \\ |E_h|^2 - |E_v|^2 \\ 2\text{Re}(E_h E_v^*) \\ -2\text{Im}(E_h E_v^*) \end{pmatrix}$$

### 瑞利散射截面
$$\sigma_{Ray} = \frac{\pi^5}{\lambda^4} |K|^2 D^6$$

其中 $|K|^2 = \left|\frac{m^2-1}{m^2+2}\right|^2$，$m$ 为复介电常数。
