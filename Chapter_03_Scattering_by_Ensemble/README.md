# 第3章：水凝物集合的散射

## 章节简介

本章讨论水凝物集合的散射特性，包括距离权重函数、粒子取向分布对散射统计特性的影响，以及极化变量的集合平均计算。

**核心主题：**
- 距离权重函数与雷达体积采样
- 粒子取向分布（倾角分布）的影响
- 后向散射协方差矩阵
- 极化变量的统计定义

## 关键公式

### 距离权重函数
$$W(r) = \exp\left(-\frac{4\ln 2 (r-R_0)^2}{\Delta r^2}\right)$$

其中 $R_0$ 是雷达距离库，$\Delta r$ 是距离分辨率。

### 后向散射协方差
$$\langle S_{hh} S_{vv}^* \rangle = \int p(\theta) S_{hh}(\theta) S_{vv}^*(\theta) d\theta$$
