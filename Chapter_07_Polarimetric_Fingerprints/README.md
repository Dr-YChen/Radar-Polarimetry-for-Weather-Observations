# 第7章：微物理过程的极化"指纹"

## 章节简介

本章介绍如何利用极化雷达变量识别降水中的微物理过程，包括碰并破碎、尺寸分选和融化层过程。

**核心主题：**
- 雨滴碰并和自发破碎过程
- 风切变导致的尺寸分选
- 融化层的亮带特征

## Notebook 讲义

| 文件 | 主题 | 交互功能 |
|------|------|----------|
| `7_1_Coalescence_Breakup_Collision.ipynb` | 碰并与破碎 | 初始滴谱、碰撞参数 |
| `7_2_Size_Sorting_Effects.ipynb` | 尺寸分选效应 | 风切变、下落速度差 |
| `7_3_Ice_Phase_Melting_Layer.ipynb` | 冰相与融化层 | 高度、温度廓线 |

## 学习目标

1. 理解碰并破碎对DSD和极化变量的影响
2. 识别尺寸分选产生的极化特征
3. 识别融化层的亮带特征
4. 理解微物理过程与极化变量的关系
