# 第6章：数据质量与测量误差

## 章节简介

本章讨论极化雷达数据的质量问题，包括定标方法、衰减校正算法和杂波识别技术。这些是获取准确极化测量结果的关键步骤。

**核心主题：**
- ZDR定标方法（天顶观测法Birdbath、干雪法）
- 衰减校正算法（PHI方法）
- 地物杂波识别
- 测量噪声与误差传播

## Notebook 讲义

| 文件 | 主题 | 交互功能 |
|------|------|----------|
| `6_1_Radar_Calibration_Simulation.ipynb` | 雷达定标模拟 | 定标方法选择、误差注入 |
| `6_2_Attenuation_Correction.ipynb` | 衰减校正算法 | 降雨率、距离衰减量 |
| `6_3_Clutter_Identification.ipynb` | 杂波识别 | 极化特征阈值设定 |

## 学习目标

1. 理解ZDR定标的物理原理
2. 掌握PHI方法进行衰减校正
3. 识别地物杂波的极化特征
4. 评估测量误差对极化变量的影响
