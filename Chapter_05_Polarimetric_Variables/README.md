# 第5章：雷达极化变量

## 章节简介

本章详细定义和讨论雷达极化变量的物理意义和计算方法，包括反射率Z、差分反射率ZDR、比差分相移KDP、共极化相关系数ρhv和退极化比LDR。

**核心主题：**
- 反射率因子Z和差分反射率ZDR的瑞利近似计算
- KDP的波长依赖性（S、C、X波段对比）
- 共极化相关系数ρhv的物理意义
- 退极化比LDR与交叉相关系数

## 学习目标

1. 掌握瑞利散射近似下Z和ZDR的计算方法
2. 理解KDP与波长的关系及其物理原因
3. 理解ρhv作为粒子一致性指标的含义
4. 学会使用极化变量识别不同降水类型

## 关键公式

### 反射率因子
$$Z = \frac{4\lambda^4}{\pi^4 |K|^2} \int_0^\infty N(D) \sigma_{bb}(D) dD$$

### 差分反射率
$$Z_{DR} = 10\log_{10}\frac{Z_{hh}}{Z_{vv}}$$
