# 第2章：极化多普勒雷达

## 章节简介

本章介绍极化多普勒雷达的基本原理，包括脉冲多普勒技术、同时收发与交替收发模式、以及多普勒频移与差分相移的关系。

**核心主题：**
- 脉冲多普勒雷达的I/Q信号处理
- SHV（同时收发）和HSHV（交替收发）模式
- 多普勒频移与粒子径向速度的关系
- H和V通道之间的差分相移

## 关键公式

### 多普勒频移
$$f_D = \frac{2v_r}{\lambda}$$

其中 $v_r$ 是径向速度，$\lambda$ 是雷达波长。

### 差分相移
$$\Phi_{DP} = \frac{2\pi}{\lambda} \int_0^R [K_h(s) - K_v(s)] ds$$

其中 $K_h$ 和 $K_v$ 是水平和垂直极化波的传播常数。
